# Description

A PulseAudio client written in pure JavaScript for [node.js](http://nodejs.org/).

Currently most commands are implemented. SHM/MemFD-related commands are not supported.

## Requirements

* [node.js](http://nodejs.org/) -- v6.0.0 or newer

## Installation
